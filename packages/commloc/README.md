# COMMLOC

## common local utilities and helpers

### img Sizes:

| Key                      | Value |
 | :-----                  | :----|
| x-sm: (extra-small)      | '50' |
| sm: (small)              | '100'|
| md: (medium **default**) | '200'|
| lg: (large)              | '400'|
| x-lg: (extra-large)      | '500'|
| jumbo: (jumbo)           | '600'|
| hero: (hero)             | '800'|

Create imgSizes using the imgSize function:
```javascript
    imgSize('jumbo') //600
```
